# 极速堆叠窗口生成器 (Stacking Window Generator)

一个基于 Python Tkinter 的极速堆叠窗口演示程序，能够快速生成彩色窗口并支持堆叠效果，具有自动安全关闭机制。

## ✨ 功能特性

* ​**⚡ 极速生成**​：采用多线程技术，实现毫秒级窗口创建速度
* ​**🎯 完全随机**​：窗口位置、大小、颜色完全随机生成
* ​**🔄 可堆叠设计**​：窗口支持重叠堆叠，创造密集视觉效果
* ​**🚫 无数量限制**​：取消窗口生成上限，持续生成直到程序结束
* ​**⏰ 安全机制**​：60秒自动停止并清理所有窗口
* ​**🎨 多彩效果**​：每个窗口拥有随机背景色和智能对比文字色
* ​**✏️ 自定义文字**​：支持用户自定义窗口显示内容
* ​**🌫️ 透明度效果**​：部分窗口具有随机透明度增强视觉效果

## 📋 系统要求

* Python 3.6+
* Tkinter（通常随Python一起安装）
* 支持图形界面的操作系统（Windows/macOS/Linux）

## 🚀 快速开始

### 安装运行

1. ​**克隆或下载项目文件**​

```
git clone <repository-url>
cd stacking-window-generator
```

1. ​**直接运行程序**​

```
python stacking_window_generator.py
```

### 使用说明

1. 程序启动后会显示声明对话框
2. 在输入框中设置自定义文字（多个文字用逗号分隔）
3. 点击"生成"按钮开始窗口生成
4. 程序会自动运行60秒后停止
5. 可随时使用 `Ctrl+C`强制退出程序

## ⚙️ 程序配置

### 自定义文字设置

* 默认文字：`test,演示,窗口,Python,Hello`
* 支持中英文混合输入
* 多个文字用逗号分隔
* 留空将使用默认文字 'test'

### 窗口参数

* ​**尺寸范围**​：宽度 200-400px，高度 60-120px
* ​**生成速度**​：批量生成，间隔 0.01秒
* ​**透明度**​：30%的窗口具有 0.7-0.9 的随机透明度
* ​**字体大小**​：随机 8-12px

## 🔧 技术实现

### 核心架构

```
class StackingWindowGenerator:
    """极速堆叠窗口生成器主类"""
    - 多线程窗口生成
    - 随机位置分配算法

    - 自动资源管理
```

### 关键技术

* ​**资源监控**​：自动清理已关闭窗口引用
* ​**异常处理**​：完善的错误处理和恢复机制

## ⚠️ 注意事项

1. ​**系统资源**​：程序会快速生成大量窗口，请确保系统有足够内存
2. ​**强制退出**​：如遇异常可使用 `Ctrl+C`强制终止
3. ​**窗口管理**​：所有窗口会在程序结束时自动关闭
4. ​**性能影响**​：生成大量窗口可能暂时影响系统响应速度

## 🆘 常见问题

​**Q: 程序无法启动？​**​

A: 检查Python版本和Tkinter支持，确保满足系统要求。

​**Q: 窗口生成太慢？​**​

A: 程序已优化生成速度，如仍感觉慢可能是系统性能限制。

​**Q: 如何修改运行时间？​**​

A: 在代码中修改 `_auto_stop`方法中的时间参数。

​**Q: 窗口堆叠太密集？​**​

A: 这是程序设计特性，如需调整可修改位置生成算法。
